# Hono Official Datastar Examples

A single Hono Node app that ports the official Datastar examples to Datastar Kit.

The Datastar Kit debugger is rendered on every page, so this example app is a convenient place to inspect live signals and searchable Datastar event activity.

```sh
pnpm --filter @datastar-kit/example-hono-official-examples dev
```

The server listens on `http://localhost:3000` by default. Set `PORT` to use a different port.
